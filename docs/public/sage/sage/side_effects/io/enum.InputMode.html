The different types of input modes a program might use.

Variants§

§

StdinChar

Standard input modes (The standard interface is typically used for command line programs, and for all pipelines between programs) Standard input (ASCII character)

§

Standard input (integer)

§

§

Special input modes A random number

§

§

Button

Input from a button (0=not pressed, 1=pressed)

§

Keyboard

Input from keyboard (ASCII character)

§

JoyStick(Direction)

Input from a JoyStick the degree of displacement in a given direction (from -128 to 128).

§

Clock

Physical sensor input modes (These should typically be implemented for devices that measure the physical world, like a smart watch or a smart phone) Time (in seconds) since the program started
